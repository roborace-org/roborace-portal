# roborace-portal

API and UI for Roborace Portal


## How to install (API and WebUI)
- Install `python3-dotenv` and dependencies
    
```bash

pip3 install -r requirements.txt

```

## How to start (API)

```bash

python3 main.py

```

## How to start (WebUI)

```bash
cd roborace-ui

./run.sh

```
